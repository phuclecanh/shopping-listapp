# Shopping List App

A simple shopping list web app built with plain HTML, CSS, and JavaScript.

## Features
- Add items by typing and pressing Enter or clicking **+**
- Check off items with a checkbox (strikethrough styling)
- Delete individual items with the **✕** button
- Bulk-remove all checked items with **Clear checked**
- Items persist across page reloads via 

## Usage
Open  directly in any browser — no server needed.
